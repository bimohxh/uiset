# 提交

clone本项目，然后在 `categorys/[分类]` 下面新建一个目录，添加下面两个文件

1、`meta.json` 用来描述资源的信息，格式如下

```json
{
    "name": "Lucide",
    "summary": "好看的SVG图标",
    "website": "https://lucide.dev/",
    "github": "https://github.com/lucide-icons/lucide",
    "tags": "标签，多个标签用英文逗号隔开",
    "isFree": "是否免费"
}
```

2、`thumbnail.png` 缩略图，推荐大小为（200 * 150 px）
